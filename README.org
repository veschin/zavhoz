#+title: Zavhoz
Утилита для хранения конфигов и паролей, а так же удобного присмотра за ними

* Список фичей
- хранение паролей
  - дата создания
  - дата изменения
- бэкап директорий и файлов
  - рекурсивные бэкапы
  - архивирование
- подсказки по структуре
  - CLI формат работы
  - информация по текущим бэкапам
